Conheça esta ferramenta poderosa ... chamada Tableau


Galera este blog aqui é para falar mais um pouco do Tableau, a versão que quero abordar é a gratuita, isso mesmo eles possuem uma versão free, que dá para poder fazer muitas tarefas ela é muito robusta, simples de configurar e para realizar tarefas de integração com banco de dados como excel é simples de realizar e você obtém resultados incríveis, consequentemente suas tarefas ficam automatizadas e a economia de tempo é excelente, os frutos desse processo faz com que os colaboradores fiquem mais produtivos, é possível integrar mais que uma base de dados e realizar configurações para se obter as tabelas necessárias, uma vez feito este processo, você pode construir sua análise com diversas ferramentas como gráficos, cálculos que podem ser customizados com sua própria configuração, ambiente que funciona em tempo real, interação e dinamismo com dados.

Podemos destacar também que o Tableau possui uma configuração de dashboard que pode ser personalizada e publicada via Web, fazendo com que de qualquer lugar você possa acessar os dados demonstrativos de suas apresentações o que facilita a dinâmica de reuniões e palestras.

Seu site disponibiliza muitos tutoriais em vídeos para quem está começando aprender a ferramenta, isso é muito bom de início, abaixo segue um exemplo de Dashboard para exemplificar o acompanhamento de agentes de atendimento em uma empresa de Call Center.
