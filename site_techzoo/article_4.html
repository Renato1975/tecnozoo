Galera Python foi criado em 1991 por Guido van Rossum, é uma linguagem de programação interpretada, orientada a objetos e de tipagem dinâmica forte e que vem em uma crescente evolução, é importante ressaltar que com poucas linhas de código e de forma simples é possível programar de uma forma dinâmica, mas em cima disso também não podemos deixar de enfatizar que é necessária dedicação nos estudos, como qualquer outra linguagem de programação para aprendê-la, finalmente chegou uma linguagem de programação "mais universal", consequentemente está ajudando e motivando muitos iniciantes a entrarem para o mundo da programação, é preciso porém ir mais além pois o python não é somente isso, pois possui uma estrutura muito robusta, onde várias empresas podemos destacar entre elas Google, Youtube, Dropbox, Pinterest, Instagram.

Também é uma linguagem de programação completa, isto é você pode realizar projetos para desktop, Mobile e Web, possui muitos frameworks de excelente qualidade que ajudam os programadores como o django para desenvolvimento Web que já vem como banco de dados SQLite3, já com a facilidade com setup Admin já pronto para o gerenciamento do seu banco de dados, onde fica extremamente muito mais simples para criar suas tabelas e configurar por esta página "adm", possui também o Flask e o Bottle para desenvolvimento Web, Tkinter para interfaces gráficas, Beautiful Soup que cria uma árvore de análise para páginas analisadas que podem ser usadas para extrair dados do HTML, o que é útil para a extração da web, o framework Dash é uma estrutura Python de código aberto usada para criar aplicativos da Web analíticos.

É especialmente bom para os cientistas de dados em Python que não estão muito familiarizados com o desenvolvimento da web, estes foram somente alguns comentados, porém o Python possui muitos frameworks que vão com certeza te ajudar em seus projetos.
